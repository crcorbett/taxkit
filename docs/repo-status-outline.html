How The Repo Works Today

Runtime Flow

  • The root route in apps/web loads health data.
  • It resolves the route runtime and yields WhatTaxHttpApiService.
  • The service calls api.health.getHealth() through the browser-safe client boundary.
  • The page renders the returned ok status.

Package Boundaries

  • apps/web owns the current app shell.
  • packages/http-api owns the current API contract.
  • packages/tsconfig owns shared TypeScript config.
  • packages/core, packages/scripts, and packages/ui currently have README coverage but no implemented package manifests.
