How the repo works today

Runtime flow

  • The root route in apps/web loads health data.
  • It resolves the route runtime and yields WhatTaxHttpApiService.
  • The service calls api.health.getHealth() through the browser-safe client boundary against apps/api.
  • The page renders the returned ok status.

Package boundaries

  • apps/api owns the current API runtime.
  • apps/docs owns the public docs runtime.
  • apps/web owns the current app shell.
  • packages/api/http owns the current API contract.
  • packages/calculators owns calculator orchestration.
  • packages/sdk/typescript owns the private SDK package.
  • packages/docs-content owns WhatTax docs contracts.
  • packages/docs-fumadocs owns reusable Fumadocs wiring.
  • packages/tsconfig owns shared TypeScript config.
  • packages/scripts owns Effect-native release gate orchestration.
  • packages/ui remains a planned README-only area.
