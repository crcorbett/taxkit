# Design

WhatTax inherits the quiet, text-led design system from `/Users/cooper/Projects/site`: Tailwind v4 tokens, Geist typography, Base UI primitives, and restrained surfaces.

## Principles

- Keep the interface dense, readable, and explicit.
- Use semantic tokens from `packages/ui/src/styles/base.css`; avoid raw color values in components.
- Prefer compact controls, mono labels for structured metadata, and clear status text.
- Cards are for repeated items or true panels, not page sections.

## Identity

The current identity is intentionally spare: `WhatTax` as a mono wordmark, neutral surfaces, and limited accent color. Revisit this file once the product direction is clearer.
