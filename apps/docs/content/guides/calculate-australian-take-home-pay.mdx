# Calculate Australian take-home pay

Use this guide when you need a pay-period take-home amount from TypeScript.
The SDK path keeps the calculation in-process and avoids HTTP transport.

## Before you start

- Install the SDK with [Install the SDK](../start/install-the-sdk.mdx).
- Use a server-side or build-time TypeScript runtime.
- Represent money in cents with `aud`.

## Steps

1. Import `aud`, `GrossPay` and the AU SDK helper.
2. Build the canonical gross pay fact.
3. Call `au.pay.takeHomePay`.
4. Handle expected failures with the safe helper when user input is not trusted.

## Example

```ts
import { aud } from "@whattax/core/primitives";
import { GrossPay } from "@whattax/rules-au-pay";
import { au } from "@whattax/sdk/au";

const report = await au.pay.takeHomePay({
  grossPay: new GrossPay({
    amount: aud(165_400),
    period: "weekly",
  }),
  taxFreeThresholdClaimed: true,
});

console.log({
  kind: report._tag,
  netPayCents: report.netPay.cents,
});
```

## Verify the result

Expected output:

```json
{ "kind": "TakeHomePayReport", "netPayCents": 130100 }
```

This matches the current SDK test fixture for weekly gross pay of $1,654.00.

## Handle errors

Use the safe helper when input crosses an untyped boundary:

```ts
const result = await au.pay.safe.takeHomePay({
  grossPay: new GrossPay({
    amount: aud(165_400),
    period: "weekly",
  }),
  taxFreeThresholdClaimed: true,
});

if (result._tag === "WhatTaxFailure") {
  console.error(result.error.message);
}
```
